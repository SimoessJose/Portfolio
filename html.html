HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de uma página web. Com suas tags e elementos, é possível definir títulos, parágrafos, imagens, links e outros componentes que compõem a estrutura lógica do documento. Em conjunto com o CSS e o JavaScript, o HTML é fundamental para criar páginas web interativas e visualmente atraentes, proporcionando uma experiência de navegação agradável para os usuários.

Prós:

  • Estruturação lógica do conteúdo da página.
  • Compatibilidade com diferentes navegadores.
  • Fácil acessibilidade para leitores de tela e motores de busca.

Contras:

  • Limitado em termos de design e interatividade sem o uso de CSS e JavaScript.
  • Pouca interatividade
  • Dificuldade em estruturar dados complexos
